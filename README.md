# Installation
1.  Install ``django-unused-media``:
    ```
    pip install django-unused-media
    ```
    Python 2.7, 3.3, 3.4, PyPy are supported<br/>
    Django 1.6, 1.7, 1.8 are supported

2.  Add ``django-unused-media`` to ``INSTALLED_APPS``:
    ```python
    INSTALLED_APPS = (
        ...
        'django_unused_media',
        ...
    )
    ```
